<!-- GETTING STARTED -->
### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/chelseaharris137/MRI-Simmons-UI-Test.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run in development mode
   ```sh
   npm start
   ```
4. Open http://localhost:3000 to view the app in the browser.
